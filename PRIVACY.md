# tiny-yt Privacy Policy

Last updated: 2026-04-17

`tiny-yt` does not collect, store, transmit, sell, or share personal information or browsing history.

What the extension accesses:

- The currently active tab, only after the user clicks the extension button
- The page's video elements, only to request or exit Picture-in-Picture for the current page

What the extension does not do:

- It does not send data to any server
- It does not use analytics or tracking
- It does not store user data locally or remotely
- It does not read tabs in the background without a user action
- It does not inject remote code

Hosting note:

If this privacy policy is hosted on GitHub Pages, GitHub may process standard web hosting logs such as IP addresses
for security and operational purposes. That hosting activity is separate from the extension itself. The extension does
not send usage or personal data to the developer.

If this behavior changes in a future release, this policy should be updated before that release is published.
