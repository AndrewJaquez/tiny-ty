# GitHub Pages

This repository now includes a small static site in `docs/` for GitHub Pages.

Recommended setup:

1. Push the repository to GitHub.
2. In repository settings, open **Pages**.
3. Set the source to **Deploy from a branch**.
4. Select the default branch and the `/docs` folder.
5. Save and wait for the site to publish.

For the current `origin` remote, the expected project-site URLs are:

- Home: `https://getbiddable.github.io/tiny-yt/`
- Privacy policy: `https://getbiddable.github.io/tiny-yt/privacy/`
