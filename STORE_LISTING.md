# tiny-yt Chrome Web Store Listing

## Summary

Open the current YouTube video in a tiny floating Picture-in-Picture window that stays on top while you work.

## Detailed Description

`tiny-yt` gives YouTube a fast one-click Picture-in-Picture shortcut.

Open a YouTube video, click the toolbar icon, and the current video pops into a small always-on-top window so you can keep watching while working in other tabs or apps. Click the icon again to close Picture-in-Picture.

Why it is useful:

- One click to open PiP on the current page
- Small floating player that stays above other windows
- No account required
- No background syncing
- No ads, tracking, or analytics added by the extension
- Minimal permissions: `activeTab` and `scripting`

## Privacy Tab Answers

Single purpose:

- Open the current page's YouTube video in Picture-in-Picture on user request

Data collection:

- No data collected
- No data sold
- No data shared with third parties
- No remote code
